# Contributing

This repository is the public support, issue-tracking, and release-notes home for Starcat Pro. It does not contain the Starcat application source code or production configuration.

- Keep `README.md` and `README-ZH.md` aligned when product-facing content changes.
- Keep `CHANGELOG.md` and `CHANGELOG-ZH.md` aligned when release notes change.
- Preserve the public `banner.webp` and `main.webp` paths because other Starcat repositories reference them directly.
- Do not commit application binaries, signing material, customer data, license keys, or private configuration.

For product bugs and feature requests, open an issue with the Starcat version, macOS version, distribution channel, reproduction steps, and screenshots when useful.
