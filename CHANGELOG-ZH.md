# Starcat 更新日志

这里记录 Starcat 的重要变化，内容用于发布说明。

## 1.0.0

Starcat 正式初始版本。

### 亮点

- 用原生 macOS 三栏界面管理 GitHub Stars。
- 支持 GitHub 登录和 Stars 同步，包含进度显示、刷新和限流提示。
- 支持按全部收藏、标签、语言、智能集合、状态、归档、Fork 和搜索浏览仓库。
- 支持查看仓库详情，包括元信息、README、GitHub 快捷入口、克隆链接和取消收藏。
- 支持标签、批量打标签、私有笔记，以及阅读或使用状态管理。
- 支持本地搜索仓库名称、作者、描述、笔记和相关元数据。
- 支持 GitHub Trending、周刊来源、推荐和仓库活动浏览。
- 支持 Release 追踪、仓库订阅、资源查看和已读状态管理。
- 支持自带配置的 AI 能力，包括仓库摘要、标签建议、对话、语义搜索、README 翻译和分享辅助。
- 支持仓库健康度、OpenSSF Scorecard、Wiki 上下文和相关洞察。
- 支持创建和导出分享卡及仓库集合。
- 支持应用设置、存储管理、诊断、开源致谢、主题、语言和界面偏好。
- 支持英文和简体中文。
- 作为原生 macOS 应用运行，包含沙盒、强化运行时、窗口管理和 App Store 上架准备。

### 新增

- 新增 Getting Started 新手引导系统，覆盖同步、RAG/Agent 试用、项目主页和知识库等关键操作，支持返回导航和调试模式开关。
- 新增浮层状态保护防止异常退出后引导入口锁死，统一手动重播防护，以及分享指引气泡位置优化。
- 新增 Agent 与 RAG 独立窗口工作台，支持窗口置顶、侧边栏折叠，并继承主窗口界面字号。
- 新增 Agent 计划与工具输出展示、扩展事件模型，并完善运行时事件反馈闭环。
- 新增落地页首屏加载动效及多阶段过渡动画、macOS 专属下载弹窗及优化后的定价网格，以及下载版本区块。
- 新增相似仓库推荐，支持保存推荐结果、展示更清晰的推荐卡片、标记已收藏仓库，并可在新的 Starcat 窗口中打开推荐仓库。
- 新增更多 GitHub 登录方式，包括浏览器登录、Token 登录、登录方式选择器和授权倒计时。
- 新增仓库代码智能分析能力，作为 Pro 功能提供独立的仓库分析工作区、结果缓存和设置入口。
- 新增详情页 AI 助手入口、README AI 摘要、摘要缓存，并支持从 Starcat 和浏览器入口触发摘要生成。
- 新增 Browser Plugin 工作流，可在 GitHub 页面与 Starcat 配对，并访问仓库上下文、笔记、标签、健康度、Wiki 上下文、推荐和 Safari WebExtension 支持。
- 新增开始使用清单，引导首次设置和关键操作。
- 新增探索发现相关页面，覆盖趋势、发现、GitHub 搜索和仓库榜单。
- 新增 README 后台预拉取和健康分预热，让仓库详情更快进入可用状态。
- 新增健康分排序、健康分颜色优化和 OpenSSF Scorecard 预热，帮助更快判断仓库质量。
- 新增菜单栏与 macOS 顶部菜单控制，常用操作更容易触达。
- 新增 Direct 分发能力，与 App Store 渠道并行，包含独立 Direct 构建目标、渠道配置和仅 Direct 版本启用的 Sparkle 自动更新。
- 新增 Direct 版 Sparkle 更新支持，包含 appcast 更新清单、EdDSA key 配置、检查更新菜单入口，以及私钥备份和多台 Mac 签名流程文档。
- 新增 Direct License 基础设施，包含 License API 服务、本地授权存储，以及同时服务 App Store 和 Direct 渠道的权益抽象。
- 新增中立的 Direct 支付网关抽象，为后续接入 Creem 等支付 provider 预留扩展点。
- 新增服务状态徽章、服务健康检查和本地运维工具，方便配置和排查问题。
- 新增界面字号档位、发布记录分页，以及发布订阅总数侧边栏展示。

- 提供 Agent 工作台，包含可审计追踪、回复样式、清晰的工具栏入口和标题栏控制。
- 提供完整的 DMG 背景、应用拖放位置和可读的 Direct 分发文件名。
- 提供优化后的落地页视觉、macOS 专属下载流程、定价布局和产品质感。
- 提供首次同步与周刊页面加载状态，以及统一加载提示组件。
- 提供推荐卡片、推荐缓存、清晰入口、已收藏标记和 Pro 权限提示。
- 提供 README 阅读体验，包含 GitHub 图片处理、系统风格滚动条和清晰加载状态。
- 提供全局搜索，包含快捷键、历史记录、可靠焦点获取和 GitHub 搜索结果分页。
- 提供 GitHub Stars 列表、侧边栏统计、语言图标和空状态展示。
- 提供 Activity 与 Weekly 浏览能力，包含筛选、快速数量统计和详情加载状态。
- 提供分享卡能力，包含多种版式、配色、个人资料展示和 Pro 可用性判断。
- 提供设置页文案、存储操作、服务配置和诊断日志反馈。
- 提供深色模式下 AI、健康度、搜索、分享和插件相关界面的完整适配。
- 提供发布准备材料、开源致谢、分发方案和应用内发布说明读取能力。
- 提供 Direct 发布工具链，包含 App Store / Direct 独立打包脚本、Direct 运行入口、Sparkle appcast 生成和 appcast / DMG 上传编排。
- 提供 Starcat 官网 nginx 规则，支持 Sparkle appcast 不缓存和 Direct 下载路径。
- 提供工具栏、危险操作、常用操作、标签、侧边栏和批量操作中的统一图标规范。
- 提供内部诊断、遥测安全边界和开发专用开关，避免未完成能力被当成正式功能。
- 提供 Agent 默认产物加载选择。
- 提供 RAG 与工作台输入框的稳定首键输入体验。
- 提供 RAG 中栏消息对齐和工作台标题栏图标显示。
- 提供正确的落地页 macOS 版本文案。
- 提供关闭主窗口后通过 Dock 或菜单栏重新打开 Starcat 的行为。
- 提供浮层覆盖 README 或其他交互区域时的正确鼠标光标行为。
- 提供 GitHub 仓库名称大小写不一致时的 AI 摘要生成能力。
- 提供浏览器插件中不可用或 Pro 限制动作的明确处理。
- 提供外部入口修改笔记和标签后的可见内容自动刷新。
- 提供推荐仓库新窗口和 README 加载的稳定体验。
- 提供 CodebaseMemory 启动、存储、缓存和仓库切换支持。
- 提供子目录 README 图片和 GitHub raw 图片路径显示。
- 提供全局搜索焦点时机和搜索结果交互细节。
- 提供存储重置完成态、未登录状态下存储页滚动和设置页布局边界处理。
- 提供语言数据缺失时的语言统计排序。
- 提供 Release 构建可用的调试菜单控制。
- 提供 Direct 构建签名和 entitlements 检查，确保非 App Store 版本不带 App Sandbox entitlement，同时 App Store 版本继续保留沙箱配置。
- 提供 Direct 更新测试打包流程，支持临时版本覆盖 marketing/build 版本，并避免 appcast 生成时混入旧 DMG。
